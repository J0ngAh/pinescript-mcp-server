Troubleshooting Tips

If you can view this page but not connect to the server, check:

  1. Are you using the correct port? Try port 8080 instead of 3000/3001
  2. Is your Windows Firewall blocking connections?
  3. Do you have any antivirus software that might be blocking connections?
  4. Try accessing via both localhost:8080 and 127.0.0.1:8080

This file is a static HTML page and doesn't require a server to view. If you can see this but not connect to the server, it indicates a server issue rather than a browser issue.
